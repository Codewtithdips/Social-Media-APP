Social Media App
Technologies: Django, HTML, CSS, Django REST Framework, Django ORM, Bootstrap

User Authentication: Implemented secure login and registration using Django REST Framework and Django ORM for database management.
Profile Customization: Allowed users to add profile images, bios, and locations. Designed UI with HTML/CSS and Bootstrap.
Feed and Search: Created a dynamic feed of posts from followed users and a search function to discover new users.
Backend Management: Utilized Django for routing, authentication, and data handling.
